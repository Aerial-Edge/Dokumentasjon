Detailed Description

A callable asyncio Datagram Protocol implementation.

For robotics programming purpose, I need this protocol does last-come-first-serve. This code is borrowed from Hilde Marie Moholt, and modified to fit ros2 by Aerial Edge.

Reference
https://docs.python.org/3/library/asyncio-protocol.html#datagram-protocols https://docs.python.org/3/library/asyncio-eventloop.html#asyncio.loop.create_datagram_endpoint https://stackoverflow.com/questions/46140556/proper-way-to-clear-an-asyncio-queue-in-python3
